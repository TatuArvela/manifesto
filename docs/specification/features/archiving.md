# Archiving

Notes can be archived to remove them from the main view without deleting them.

## Behavior

- Archive a note from the note card or editor toolbar
- Archived notes disappear from the main Notes view
- Archived notes are accessible from the Archive view in the sidebar
- Restoring an archived note returns it to the main view
- Archiving does not affect the note's content, tags, color, or other metadata
- Archiving sets `archived: true` on the note (see [Data Model](../data-model.md))
